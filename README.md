# Original Components 原创/改进组件
This includes some costumed or improved UI Components and Plugins which are not covered by standard frameworks, most of them based on Bootstrap 3/4 and PHP Laravel 5.5. Some of them are improved based on some previous authors, I have credited below.
这部分是个人原创或改进的组件

## Front End
1. Radio and Checkbox 自主定制多选/单选按钮 (HTML, CSS)  
Most of radios and checkboxs based on Bootstrap can't be customed easily, this type radio/checkbox can allow users click them just like normal buttons, developers can easily style them via change label's style.

[Source Code](/source/radio-and-checkbox.html)

2. Phone Number Format 自动标准格式化电话号码 (JS)  
The US Phone number like (111)222-3333 is easier to reconginze. This JS and PHP Functions can help developer quick format phone number.

[Source Code](/source/phone-format)

3. Auto Slug 自动添加Blog标题对应URL/Slug (JS)
